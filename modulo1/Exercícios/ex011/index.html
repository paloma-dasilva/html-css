Imagens Dinâmicas

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do navegador.

A tag PICTURE permite criar regras condicionais, varias "sources" de uma imagem de acordo com o tamanho da tela.


Reproduzindo Áudio

Utilize a tag "audio source" para testar diferentes tipos de arquivo do audio a ser reproduzido. Assim como nas imagens dinâmicas, a ordem importa.

Aviso: o tipo WAV é o mais pesado, melhor evitar.
